The following Apache projects are celebrating their birthdays in July.

## Apache Fory (1 year)

**Established:** 2025-07

**Description:** The mission of Apache Fory is the creation and maintenance of software related to fast multi-language serialization framework powered by JIT and zero-copy

**Project Homepage:** http://fory.apache.org

---

## Apache ShenYu (4 years)

**Established:** 2022-07

**Description:** The mission of Apache ShenYu is the creation and maintenance of software related to a Java native API Gateway for service proxy, protocol conversion and API governance

**Project Homepage:** https://shenyu.apache.org

---

## Apache Pinot (5 years)

**Established:** 2021-07

**Description:** The mission of Apache Pinot is the creation and maintenance of software related to distributed OLAP data store to provide Real-time Analytics to power wide variety of analytical use case

**Project Homepage:** https://pinot.apache.org

---

## Apache APISIX (6 years)

**Established:** 2020-07

**Description:** The mission of Apache APISIX is the creation and maintenance of software related to a cloud-native microservices API gateway

**Project Homepage:** https://apisix.apache.org

---

## Apache OpenWhisk (7 years)

**Established:** 2019-07

**Description:** The mission of Apache OpenWhisk is the creation and maintenance of software related to a platform for building serverless applications with functions

**Project Homepage:** https://openwhisk.apache.org/

---

## Apache MADlib (9 years)

**Established:** 2017-07

**Description:** The mission of Apache MADlib is the creation and maintenance of software related to a scalable, Big Data, SQL-driven machine learning framework for Data Scientists

**Project Homepage:** https://madlib.apache.org/

---

## Apache Kudu (10 years)

**Established:** 2016-07

**Description:** The mission of Apache Kudu is the creation and maintenance of software related to a distributed columnar storage engine built for the Apache Hadoop ecosystem.

**Project Homepage:** https://kudu.apache.org/

---

## Apache NiFi (11 years)

**Established:** 2015-07

**Description:** The mission of NiFi is the creation and maintenance of software related to Easy to use, powerful, and reliable system to process and distribute data

**Project Homepage:** https://nifi.apache.org/

---

## Apache Celix (12 years)

**Established:** 2014-07

**Description:** The mission of Celix is the creation and maintenance of software related to Implementation of the OSGi specification adapted to C

**Project Homepage:** https://celix.apache.org/

---

## Apache Tez (12 years)

**Established:** 2014-07

**Description:** Apache Tez is an effort to develop a generic application framework which can be used to process arbitrarily complex DAGs of data-processing tasks and also a re-usable set of data-processing primitives which can be used by other projects.

**Project Homepage:** https://tez.apache.org/

---

## Apache JSPWiki (13 years)

**Established:** 2013-07

**Description:** The mission of JSPWiki is the creation and maintenance of software related to Leading open source WikiWiki engine, feature-rich and built around standard J2EE components (Java, servlets, JSP).

**Project Homepage:** https://jspwiki.apache.org/

---

## Apache Steve (14 years)

**Established:** 2012-07

**Description:** The mission of Steve is the creation and maintenance of software related to Apache's Python based single transferable vote software system

**Project Homepage:** https://steve.apache.org/

## Apache DB (24 years)

**Established:** 2002-07

**Description:** The mission of the Apache DB project is to create and maintain commercial-quality, open-source, database solutions based on software licensed to the Foundation, for distribution at no charge to the public.

**Project Homepage:** https://db.apache.org/
